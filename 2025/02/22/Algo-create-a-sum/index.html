Problem Breakdown

We need to implement a sum function that:

  1. Supports chainable calls, such as sum(1)(2)(3).
  2. Returns the accumulated sum when no further arguments are provided.
  3. Should be implicitly convertible to a number, so sum(1)(2) == 3 evaluates to true.
