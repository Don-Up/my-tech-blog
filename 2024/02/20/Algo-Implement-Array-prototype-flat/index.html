This problem requires implementing a custom flat function to flatten a nested array to a specified depth. The input consists of an array arr and an optional integer parameter depth, which defaults to 1, meaning it flattens one level of nesting. The function returns a new array that is the flattened version of the input. For example, flat([1, [2], [3, [4]]], 1) returns [1, 2, 3, [4]], while flat([1, [2], [3, [4]]], 2) returns [1, 2, 3, 4].

Explanation

  • Handling Depth:
    The function checks the depth parameter. If depth is greater than 0, it flattens the array recursively; if depth is 0, it simply returns the original array.
